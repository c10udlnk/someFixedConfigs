- `IDA_cfg`：个性化修改IDA pro（7.0）的配置文件（具体见[【从无到有】配置IDA | c10udlnk_Log](https://c10udlnk.top/2020/11/10/toolsOf-IDAold/)），以符合个人的逆向分析习惯。
- `live2d_cfgWithHexo`：针对Melody主题进行调整的`live2d-widget`配置。
